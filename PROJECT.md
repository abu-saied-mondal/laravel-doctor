# Laravel Doctor

Laravel Doctor is an open-source diagnostics and repair toolkit for Laravel applications.

Mission

Diagnose.
Explain.
Repair.
Optimize.

Goals

• Detect Laravel issues.
• Explain why they happen.
• Repair common issues safely.
• Generate reports.
• Validate production deployments.

Target Users

• Laravel Developers
• Freelancers
• Agencies
• Startups
• DevOps Engineers

Not a cache cleaner.

A Laravel health platform.

This is your North Star.
